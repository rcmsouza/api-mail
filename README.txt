README

API de envio de e-mail, utilizando mailer fake.

- Clonar projeto
- Rodar Yarn ou npm install - para instalar dependencias
- yarn typeorm migration:run - para rodas as migrations e criar o banco de dados (sqlite)
- yarn dev - para iniciar o projeto em localhost:333


Tecnologias:

NodeJS (Typescript)
Express
Typeorm
Jest
SQLite
